## <div align="center">💻 Kodlama Standartları</div>

<table align="center">
  <tr>
    <td>
      <h3>🔍 Norminette Kuralları</h3>
      <ul>
        <li>Fonksiyonlar 25 satırdan uzun olmamalı</li>
        <li>Her satır 80 karakterden uzun olmamalı</li>
        <li>Her fonksiyon maksimum 4 parametre alabilir</li>
        <li>Maksimum 5 değişken bildirilmeli</li>
        <li>Tüm değişkenler alt çizgi (_) ile başlamalı</li>
        <li>Her fonksiyon <code>ft_</code> öneki ile başlamalı</li>
      </ul>
    </td>
    <td>
      <h3>📝 Kod Stili</h3>
      <ul>
        <li>Header dosyasında <code>ifndef</code> koruması kullanılmalı</li>
        <li>Global değişkenler yasaklanmış</li>
        <li>Tüm fonksiyonlar uygun şekilde belgelendirilmeli</li>
        <li>Yorum satırları anlamlı ve açıklayıcı olmalı</li>
        <li>Kod mantıklı birimler halinde düzenlenmeli</li>
        <li>Değişken isimleri anlamlı olmalı</li>
      </ul>
